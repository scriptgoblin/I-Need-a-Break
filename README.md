## What it does

Opens a progress bar that cycles through real filenames from a folder you choose — at exactly the right pace to fill your entire break duration. Looks like work. Feels like rest.

Optional **Lock Editor** mode covers the Unity window and sets the cursor to busy so you can step away without accidentally clicking anything.

## Installation

Drop `INeedABreak.cs` anywhere inside your `Assets/` folder. That's it — no package, no dependencies.

> **Tip:** add it to your `.gitignore` so your teammates don't have to know. The file can be renamed to anything — it won't affect how it works.

## Usage

**Help → I Need A Break → Take A Break**

> Hidden under the Help menu — because nobody ever checks the Help menu.

- Pick a folder (defaults to `Assets/`)
- Set your duration
- Optionally enter a custom progress message
- Toggle **Lock editor during break** (on by default)
- Hit **Start Break**

To cancel early: **Help → I Need A Break → Stop Break** or `Cmd+Alt+B` (Mac) / `Ctrl+Alt+B` (Win)
